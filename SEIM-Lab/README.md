## What is SIEM?
SIEM is a system that collects, analyzes, and correlates logs from multiple sources to detect suspicious activity and security threats.

## Real-World Relevance
SIEM tools like Splunk and IBM QRadar are used in Security Operations Centers (SOC) to monitor networks and detect cyber threats in real-time.
